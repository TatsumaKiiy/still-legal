Politique de confidentialité

Vos données, en toute transparence.

Dernière mise à jour : 28 avril 2026

En résumé : Still fonctionne 100% en local sur votre appareil. Aucune donnée personnelle n'est collectée par défaut. Aucun email n'est demandé, aucun compte n'est requis. Nous proposons un toggle optionnel pour partager des statistiques d'usage anonymes (désactivé par défaut, activable dans les Paramètres) qui nous aide à améliorer l'application.

1. Qui sommes-nous ?

Still est une application de micro-méditation contextuelle développée par un éditeur indépendant basé en France. Ce document explique comment nous traitons vos données personnelles conformément au Règlement Général sur la Protection des Données (RGPD).

Responsable du traitement : l'éditeur de Still — nous contacter à stillapp.contact@gmail.com

2. Quelles données collectons-nous ?

2.1 Données stockées uniquement sur votre appareil

Ces informations restent sur votre téléphone et ne sont jamais transmises à nos serveurs :

  • Votre prénom (saisi pendant l'onboarding, optionnel)
  • Vos objectifs d'utilisation et vos contextes favoris
  • Votre historique de sessions (dates, durées, protocoles)
  • Votre série (streak) et vos statistiques personnelles
  • Vos préférences (son, chimes, notifications, horaires, fond visuel)

Ces données sont stockées via AsyncStorage (le système de stockage local d'iOS). Elles disparaissent si vous désinstallez l'application ou si vous utilisez la fonction « Effacer toutes mes données » dans les Paramètres.

2.2 Statistiques d'usage anonymes (opt-in désactivé par défaut)

Vous pouvez choisir de nous partager des statistiques d'usage anonymes pour nous aider à améliorer Still. Ce partage est désactivé par défaut. Pour l'activer, vous devez explicitement allumer le toggle « Partager des analytics anonymes » dans les Paramètres de l'application.

Tant que vous n'avez pas activé ce toggle, aucune donnée ne quitte votre appareil. Le SDK PostHog n'est même pas initialisé.

Si vous activez le toggle, voici ce que nous collectons via PostHog :

  • Les écrans consultés (accueil, séances, profil, paramètres)
  • Les actions effectuées (lancement de session, complétion, sélection d'objectifs)
  • Les événements liés aux abonnements Still+ (affichage du paywall, souscription, restauration)
  • Les erreurs techniques (pour identifier et corriger les bugs)
  • Des informations techniques anonymes (modèle d'appareil, version d'OS, langue)

Ce que nous NE collectons JAMAIS via les analytics :

  • Votre prénom
  • Aucune adresse email (nous n'en demandons pas)
  • Le contenu des champs texte que vous saisissez
  • Des enregistrements vidéo ou audio de vos sessions
  • Votre adresse IP complète (anonymisée par PostHog)

Vous pouvez désactiver le toggle à tout moment dans les Paramètres. Aucune donnée ne sera alors plus envoyée.

2.3 Achats in-app (Still+)

Si vous souscrivez à un abonnement Still+, la transaction est traitée par Apple StoreKit (le système d'achat intégré d'iOS). Apple est seul responsable du traitement de votre paiement. Nous n'avons jamais accès à votre carte bancaire ni à votre Apple ID.

Pour gérer vos abonnements (vérifier leur statut actif, restaurer un achat sur un nouvel appareil), nous utilisons RevenueCat, qui reçoit un identifiant anonyme technique généré par Apple, ainsi que les informations de transaction associées à votre abonnement (date, montant, statut). RevenueCat ne reçoit aucune donnée personnelle vous identifiant.

2.4 Données que nous ne collectons PAS

  • Aucune adresse email (nous n'en demandons pas)
  • Aucun nom de famille
  • Votre localisation
  • Vos contacts
  • Vos photos ou vidéos
  • Votre historique de navigation
  • Votre identifiant publicitaire (IDFA / GAID)
  • Aucune donnée biométrique ou de santé
  • Aucune information bancaire (gérée exclusivement par Apple)

3. Où sont stockées vos données ?

Les données sur votre appareil restent exclusivement chez vous. Si vous activez les analytics anonymes, les données sont stockées chez nos sous-traitants techniques, tous localisés dans l'Union européenne :

  • PostHog Cloud EU (Allemagne, UE) — pour les statistiques d'usage anonymes (uniquement si vous avez opt-in)
  • RevenueCat (États-Unis) — pour la gestion des abonnements (transactions Apple uniquement, aucune donnée personnelle)

RevenueCat est conforme aux clauses contractuelles types de l'UE pour les transferts de données. Les données traitées sont strictement techniques (statut d'abonnement) et ne contiennent aucune information personnelle vous identifiant.

4. Pourquoi collectons-nous ces données ?

Les données locales servent uniquement à faire fonctionner l'application (personnaliser l'accueil, maintenir votre série, afficher vos statistiques personnelles).

Les statistiques d'usage anonymes (si vous les avez activées) servent à :

  • Comprendre quelles fonctionnalités sont utilisées ou non
  • Identifier et corriger les bugs
  • Améliorer l'expérience utilisateur sur la base de données réelles, pas d'hypothèses
  • Mesurer l'efficacité des améliorations apportées

Ces analytics reposent sur la base légale du consentement explicite (article 6.1.a du RGPD), via le toggle activable et désactivable à tout moment dans les Paramètres.

Les données techniques d'abonnement servent uniquement à valider votre accès à Still+ et à permettre la restauration de vos achats sur un nouvel appareil. Elles reposent sur la base légale de l'exécution du contrat (article 6.1.b du RGPD).

5. Combien de temps gardons-nous vos données ?

Les données locales restent sur votre appareil tant que l'application est installée. Si vous désinstallez Still ou utilisez la fonction « Effacer toutes mes données » dans les Paramètres, elles sont supprimées définitivement.

Les statistiques d'usage anonymes (si activées) sont conservées 12 mois maximum, puis agrégées sous forme de statistiques impersonnelles.

Les données techniques d'abonnement sont conservées tant que votre abonnement est actif, plus la durée légale de conservation des informations comptables (10 ans).

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données
  • Droit de rectification : corriger des informations inexactes
  • Droit à l'effacement : demander la suppression de vos données (immédiate via le bouton dans les Paramètres)
  • Droit de limitation : restreindre le traitement de vos données
  • Droit à la portabilité : recevoir vos données dans un format structuré
  • Droit d'opposition : refuser certains traitements (notamment les analytics, désactivables dans l'app)
  • Droit de retirer votre consentement à tout moment

Pour exercer ces droits, contactez-nous à stillapp.contact@gmail.com. Nous répondons sous 30 jours maximum.

Si vous estimez que vos droits ne sont pas respectés, vous pouvez déposer une réclamation auprès de la CNIL.

7. Sécurité

Nous appliquons des mesures techniques et organisationnelles adaptées pour protéger vos données :

  • Connexions chiffrées (HTTPS / TLS) pour toutes les transmissions
  • Aucun serveur backend exposant des données utilisateurs (architecture 100% locale)
  • Pas de partage avec des tiers hors sous-traitants techniques (PostHog, RevenueCat, Apple)
  • Aucune revente de données, jamais

8. Cookies et traceurs

L'application Still n'utilise aucun cookie ni traceur publicitaire. Nous n'intégrons pas de SDK tiers de tracking commercial (Facebook, Google Analytics, AppsFlyer, etc.).

Si vous activez les analytics, PostHog utilise un identifiant anonyme technique stocké localement pour distinguer les sessions entre elles, mais ne permet pas de vous identifier personnellement.

Ce site web (pages légales) n'utilise pas non plus de cookies.

9. Enfants et mineurs

Still est destiné aux personnes de 13 ans et plus. Nous ne collectons pas sciemment de données de mineurs de moins de 13 ans. Si vous êtes parent et pensez que votre enfant nous a fourni ses informations, contactez-nous à stillapp.contact@gmail.com pour suppression immédiate.

10. Modifications de cette politique

Cette politique peut évoluer. Les changements significatifs seront notifiés via l'application. La date de dernière mise à jour est indiquée en haut de cette page.
